## Regulatory & Billing Alignment

To preserve absolute legal compliance and prevent insurance audit failures, this repository adheres to two strict rules:

### 1. Honest Coding Practice
The clinic explicitly rejects deceptive billing mechanisms. This protocol must never be billed using a National Drug Code (NDC) meant for synthetic pharmaceuticals. It must be billed as a customized clinical compound or specialized provider supply via **CPT Code 99070** or unlisted biological codes (**J3590 / A4649**).

### 2. DSHEA Boundary Adherence
While clinically potent, this compound is managed exclusively as an internal, customized biological supply. Outpatient bottles must bear the mandatory federal disclaimer box and focus strictly on structure/function claims:

> **This statement has not been evaluated by the Food and Drug Administration. This product is not intended to diagnose, treat, cure, or prevent any disease.**
